# local-p2p-chat

For our fun session on 10.02.23!

## Requirements

* NodeJS environment (works only from `v17`)
* Running mDNS services (Bonjour, Avahi, etc. - you probably have that already)
* Wireshark (to have fun)

## Usage

```bash
# Install dependencies
npm install

# Run it!
npm start
```
